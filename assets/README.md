# Finstability Assets

This folder contains app assets for Expo:

## Required Files
You need to add the following image files:

- `icon.png` - App icon (1024x1024 px recommended)
- `splash-icon.png` - Splash screen icon (512x512 px recommended)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024 px)
- `favicon.png` - Web favicon (32x32 px or 48x48 px)

## Quick Setup
You can generate placeholder icons quickly using any image editor or online tools like:
- https://icon.kitchen
- https://appicon.co
- https://easyappicon.com

The app will work without these files during development, but you'll need them for production builds.
